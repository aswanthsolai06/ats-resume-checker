# ATS Resume Checker

A modern AI-inspired Resume Analyzer built using React and Vite.

## Features

* ATS Score Analysis
* Skill Suggestions
* Missing Keyword Detection
* Responsive UI
* Local Resume Processing

## Tech Stack

* React
* Vite
* JavaScript
* CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
