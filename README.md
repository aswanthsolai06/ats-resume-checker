# ATS Resume Checker

A modern Resume Analyzer built using React and Vite.

## Features

- ATS Score Analysis
- Skill Suggestions
- Missing Keyword Detection
- Responsive UI
- Modern Design

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Run Locally

npm install

npm run dev

## Build

npm run build# ATS Resume Checker
